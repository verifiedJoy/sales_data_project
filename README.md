# Sales Performance Dashboard

## Student Project – ARVYS Technologies

This project was completed as part of the **Data Analytics Training Program at ARVYS Technologies**.

The objective of this project is to demonstrate the complete data analysis workflow, from raw data preparation and cleaning to business analysis and dashboard development using Microsoft Excel.

---

## Project Structure

### 1. Raw Dataset

**File:** `messy_data.xlsx`

This workbook contains the original dataset before any cleaning, transformation, or validation was performed.

---

### 2. Main Project Workbook

**File:** `Sales Data Project.xlsx`

This workbook contains the following worksheets:

#### Cleaned Data

The raw dataset was cleaned to improve data quality and prepare it for analysis.

Tasks performed include:

- Removing duplicate records
- Correcting inconsistent text values
- Handling missing values
- Fixing date-related issues
- Standardizing data formats

#### Validated Data

The cleaned dataset was validated to ensure accuracy and consistency.

Validation checks include:

- Date validation
- Missing value checks
- Duplicate record checks
- Data type verification
- Consistency testing

#### Analysis

Business analysis was carried out using Pivot Tables and calculated metrics to generate insights from the dataset.

#### Dashboard

An interactive dashboard was created to visualize key business metrics and support data-driven decision-making.

Dashboard features include:

- Revenue KPIs
- Sales Trend Analysis
- Regional Performance Analysis
- Product Performance Analysis
- Interactive Slicers
- Dynamic Charts

## Student Reflection

This project provided practical experience in the complete data analysis process, from handling raw data to creating an interactive dashboard.

Throughout the project, I learned how to use **Find and Replace (Ctrl + H)** to quickly identify and correct inconsistencies within a dataset. I also gained hands-on experience with **Power Query**, where I performed deep data cleaning and transformation to improve data quality and prepare the dataset for analysis.

A key part of the project involved **data validation**, which helped me ensure that the data was accurate, consistent, and reliable before conducting any analysis. I further developed my analytical skills by using **Pivot Tables** to automate data analysis, summarize large datasets, and uncover meaningful patterns and trends.

Finally, I learned how to design and build **Key Performance Indicators (KPIs)** and interactive dashboards that transform raw data into clear, actionable insights through effective data visualization.

Overall, this project strengthened my skills in data cleaning, validation, analysis, and visualization, while demonstrating the importance of a structured workflow in delivering reliable business insights.
